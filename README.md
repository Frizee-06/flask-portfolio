# Flask Portfolio Website

This is a mini project for Python Developer Internship - Task 6.

## Features
- Personal intro
- Contact form using Flask
- HTML, CSS, and Flask Routing
- Jinja2 template support

## How to Run
1. Install Flask:

pip install flask

2. Run:

python app.py

3. Open browser at http://127.0.0.1:5000/
